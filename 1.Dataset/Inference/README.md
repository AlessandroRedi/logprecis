# Cyberlab

This folder contains the data from the [CyberLab honeynet dataset](https://zenodo.org/record/3687527#.YmEr9pJBxQL).

To group the files from the `.\Chunks` folder, run:

```shell
cat ./Chunks/chunk* > all_files.csv
```
